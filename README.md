# Install
```bash
git clone git@github.com:attiand/my-bash-env.git
git clone https://github.com/juven/maven-bash-completion.git
```

Add the following to `.bashrc`

```bash
# home, work, remote
MY_LINUX_ENV_TYPE='work'
source ~/git/my-bash-env/setup.sh
```
## Configure
* Bash prompt
* Maven completion
